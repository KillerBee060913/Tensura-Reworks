# Tensura Reworks

A Minecraft Pocket Edition addon inspired by **"That Time I Got Reincarnated As A Slime"** anime.

## Features

### Core Systems
- **Magicule System**: Collect and manage magicules to power your abilities
- **Aura System**: Build and maintain your aura for enhanced power
- **EP (Energy Points) System**: Track and use energy for skill activation

### Skills
The addon includes 5 common skills:
1. **Slime Strike** - Basic melee attack with magicule boost
2. **Predation** - Consume nearby entities to gain resources
3. **Regeneration** - Restore health using aura
4. **Mimicry** - Copy properties of absorbed materials
5. **Thought Acceleration** - Temporarily increase speed and reflexes

## Installation

1. Download the addon from the repository
2. Extract both packs (Behavior Pack and Resource Pack)
3. Open Minecraft Pocket Edition
4. Go to Settings > Packs > Add Packs
5. Select both the Behavior Pack and Resource Pack
6. Create a new world and activate the packs
